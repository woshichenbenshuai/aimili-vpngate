## 中文

### 🚀 快速开始

在您的 **Ubuntu** 机器上，复制并运行以下一行指令即可完成自动安装部署：

```bash
bash <(curl -Ls https://raw.githubusercontent.com/baoweise-bot/aimili-vpngate/main/install.sh)
```
---

### 🛠️ 快捷命令行 (CLI)

安装成功后，系统会在全局注册 `ml` 快捷管理指令：
- **`ml status`** 或 **`ml`**：查看当前运行状态（监听端口、活动 VPN 节点等）。
- **`ml start`**：启动 AimiliVPN 网关服务。
- **`ml stop`**：停止 AimiliVPN 网关服务。
- **`ml restart`**：重启服务。
- **`ml logs`**：查看实时的 Systemd 服务运行日志。
- **`ml uninstall`**：完全卸载服务并清理相关环境。

### 🛠️ CLI Helper Commands

Once installed, use the global command `ml` to manage the gateway:
- **`ml status`** or **`ml`**: Check running system status.
- **`ml start`**: Start the gateway service.
- **`ml stop`**: Stop the gateway service.
- **`ml restart`**: Restart the service.
- **`ml logs`**: View real-time output logs.
- **`ml uninstall`**: Completely remove the service and repository files from your VPS.
